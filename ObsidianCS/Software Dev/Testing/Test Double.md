
## What are the [[Test Double]]?

When we are [[Testing]] some [[software]] we need to replace some [[components]] in the test. In those moments we use the [[Test Double]] to give a specific input, specific behavior, etc...

## Type of Test Double

1. [[Stubs]]
2. [[Mock]]
3. [[Fakes]]
4. [[Dummies]]
5. [[Spies]]
