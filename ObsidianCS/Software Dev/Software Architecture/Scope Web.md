## What is a [[Scope Web]]?

It's the **scope** (alcance) could have some [[web applications]] like the development, variables and functions. Can be understood like the **Architecture** used on the web

## Objects in a [[Scope Web]]
* [[Instances]] 
* [[Cluster]]
* [[Load balancer]]
